# Baseline Research - 표준화된 암호화 알고리즘 실험 구조

## 📋 **개요**

이 폴더는 모든 암호화 알고리즘 및 데이터셋 실험의 **표준 베이스라인**을 정의합니다.  
**FINAL_WADI_HMAC.md**를 기준으로 일관된 실험 방법론과 문서화 형식을 제공하여, 알고리즘 간 공정한 비교와 체계적인 성능 분석을 가능하게 합니다.

## 🎯 **베이스라인 원칙**

### **일관성 (Consistency)**
- 모든 실험에서 동일한 16개 조건 사용
- 표준화된 측정 지표 및 분석 방법
- 통일된 문서 구조 및 형식

### **비교 가능성 (Comparability)**  
- 동일한 데이터셋과 실험 환경
- 표준화된 시각화 및 분석 방법
- 객관적이고 공정한 성능 비교

### **재현성 (Reproducibility)**
- 상세한 실험 설정 및 매개변수 문서화
- 체계적인 체크리스트 및 검증 절차
- 표준화된 파일 구조 및 명명 규칙

## 📁 **파일 구조**

```
baseline_research/
├── README.md                           # 이 파일
├── BASELINE_EXPERIMENT_STRUCTURE.md    # 전체 베이스라인 구조 상세 문서
├── TEMPLATE_FINAL_DATASET_ALGORITHM.md # 결과 문서 템플릿
└── EXPERIMENT_CHECKLIST.md            # 실험 수행 체크리스트
```

## 📊 **표준 실험 구조**

### **🔢 고정 요소 (모든 실험 공통)**

- **16개 조건**: 1,10,50,100 센서 × 1,2,10,100 Hz
- **각 조건당**: 1,000개 요청
- **4개 Phase**: 체계적인 실험 단계 구분
- **9개 측정 지표**: 성공률, 검증률, 각종 시간 측정
- **서버**: 192.168.0.11:8085 (엔드포인트만 변경)

### **🔄 가변 요소 (실험별 변경)**

- **데이터셋**: WADI, HAI, 기타 ICS 데이터셋
- **알고리즘**: HMAC, RSA, AES-GCM, ECDSA, ED25519 등
- **알고리즘 특화 파라미터**: 키 크기, 암호화 방식 등

## 📄 **표준 문서 형식**

### **필수 생성 문서**

1. **`FINAL_[DATASET]_[ALGORITHM].md`** - 완전한 실험 결과 문서
   - 실험 설계서
   - 16개 조건 상세 결과 테이블  
   - 핵심 발견사항 (보안성/성능/효율성)
   - 결론 및 권장사항

2. **시각화 차트 10개** (영어/한국어 각 5개)
   - 처리 시간 분해 분석
   - RPS 성능 분석
   - 확장성 히트맵  
   - 효율성 분석
   - 종합 대시보드

3. **결과 데이터 파일**
   - CSV 형식 상세 결과
   - 텍스트 형식 요약
   - 타임스탬프 기반 파일명

## 🚀 **사용 방법**

### **1. 새 실험 시작 전**
```bash
# 베이스라인 구조 확인
cd baseline_research/
cat BASELINE_EXPERIMENT_STRUCTURE.md

# 체크리스트 준비

```

### **2. 실험 설계**
```bash

# [DATASET], [ALGORITHM] 등 플레이스홀더를 실제 값으로 치환
```

- 체크리스트를 따라 단계별 실행

- 각 Phase 완료 후 중간 결과 저장

- 5개 표준 시각화 차트 생성
