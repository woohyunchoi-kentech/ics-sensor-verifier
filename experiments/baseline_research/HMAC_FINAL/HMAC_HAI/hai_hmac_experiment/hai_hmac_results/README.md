# HAI HMAC 실험 결과

## 📊 실험 개요
- **데이터셋**: HAI (Hardware-in-the-loop Augmented ICS)
- **알고리즘**: HMAC-SHA256 인증
- **실험 조건**: 16개 조건 (1,10,50,100 센서 × 1,2,10,100 Hz)
- **총 요청**: 16,000개
- **실험 일시**: 2025-09-01

- **실험 시간**: 116.3분

- **최대 RPS**: 3,223.1
