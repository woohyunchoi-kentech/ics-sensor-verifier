## 📌 **특별 참고사항**

### **BulletProofs 특성**
- 영지식 증명으로 센서 데이터 프라이버시 보장
- 고정 크기 증명 (1395 bytes)
- Commitment 생성 → Proof 생성 → 검증 3단계
- HMAC 대비 높은 계산 복잡도

### **예상 성능 (HAI 실험 기준)**
- 1센서 1Hz: ~75ms 총 소요시간
- 100센서 100Hz: ~4ms 총 소요시간
- CPU 사용률: 2-20%
- 메모리 사용: 170-910MB

### **주의사항**
- 서버 검증 시간이 0으로 나올 수 있음 (고주파수 조건)
- 네트워크 지연이 주요 병목 구간
- 센서 수 증가 시 선형적 성능 향상 예상
